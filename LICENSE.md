### Issue: Features Not Responding

**Symptoms:** Hotkeys don't trigger expected actions

**Solutions:**
1. Confirm keybinds aren't conflicting with other software
2. Check that the target application has window focus
3. Restart both the executor and target application

5. Try resetting to default configuration

### Issue: Performance Degradation

**Symptoms:** Lag, stuttering, or high CPU usage

**Solutions:**
1. Lower precision settings to reduce computational load
2. Close background applications consuming resources

**Solutions:**
